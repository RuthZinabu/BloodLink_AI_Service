ℹ️ API Information

API Base URL: https://bloodlink-ai-service.onrender.com

Monthly Forecast Endpoint: GET /forecast/monthly

Yearly Forecast Endpoint: GET /forecast/yearly

Query Parameters:

  • blood_type - Optional: Blood type filter (e.g., O+, A-, B+, AB+, etc.)
  • component_type - Optional: Component type filter (e.g., Whole Blood, Packed Red Cells, etc.)
  • months_ahead - For monthly forecast (1-24, default 12)
  • years_ahead - For yearly forecast (1-10, default 3)

Example Request:

/forecast/monthly?blood_type=O%2B&component_type=Packed%20Red%20Cells&months_ahead=12

/forecast/yearly?blood_type=A-&years_ahead=5
